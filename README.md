# 이미집 (ImgZip)

> 올리기 좋게, 가볍게 · 위치·기기정보는 쏙 빼고

온라인 업로드용 **이미지 최적화 데스크톱 도구**입니다. 쇼핑몰 운영자, AI 이미지 생성 사용자, 디자이너·블로거가 **화질은 지키면서 용량만 확 줄여** 여러 장을 한 번에, 간단하게 처리할 수 있도록 만들었습니다.

다크 톤 기반의 절제된 UI에 단일 민트 포인트 컬러를 사용했고, Electron으로 동작합니다.

---

## ✨ 주요 기능

- **드래그앤드롭 일괄 처리** — 이미지·폴더를 끌어다 놓으면 이미지만 자동 선별 (비이미지 자동 제외)
- **용도별 프리셋** — SNS 업로드 / 쇼핑몰 상세 / 썸네일 / 원본 화질 유지 / 직접 설정
- **쇼핑몰 플랫폼 맞춤** — 스마트스토어(1000px·JPG) / 쿠팡(1000px·JPG) / 카페24(640px·자유), 각 마켓 권장 규격 자동 적용 (마켓은 WebP 미지원 → JPG)
- **WebP·JPG·PNG 변환** + 해상도 상한 + 품질 조절
- **개인정보(EXIF) 제거** — 촬영 위치·날짜·기기정보를 지워 안전하게 공유
- **화질 비교 미리보기** — 원본 vs 결과를 좌우 슬라이더로 비교
- **정직한 결과 카드** — 실제 절감률(소수점 한 자리, 100% 과장 없음), 절감 막대, 일괄/포맷/개인정보 처리 요약
- **사진별 선택/제외**, 실시간 예상 절감량, 처리 중 취소
- **예외 처리** — 손상 파일 건너뛰기, 이미 가벼운 이미지는 원본 유지

> 모든 처리는 **PC 내부(로컬)에서만** 이루어지며, 이미지나 개인정보를 외부로 전송하지 않습니다.

## 🖥 실행 방법

```bash
npm install
npm start
```

또는 [run.bat](run.bat) 더블클릭.

## 📦 설치 파일(.exe) 빌드

```bash
npm run dist
```

`dist/이미집-설치-<버전>.exe` 가 생성됩니다 — 이용약관 동의 → 설치 폴더 선택 → 진행률 → 완료(실행)로 이어지는 정식 설치 마법사(NSIS) 입니다.

## 🛠 기술 스택

- **Electron** — 데스크톱 셸
- **Canvas API** — 리사이즈 및 WebP/JPEG/PNG 인코딩 (EXIF 자동 제거)
- **electron-builder** — Windows NSIS 인스톨러
- **Pretendard** — 본문 글꼴 (로컬 내장)

## 📁 구조

| 파일 | 역할 |
